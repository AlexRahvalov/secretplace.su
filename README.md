# SecretPlace.su - Сайт для Minecraft сервера

Проект сайта для Minecraft сервера с динамически обновляющейся информацией и собственной реализацией компонентов.

## Структура проекта

```
secretplace.su/
├── client/                 # Фронтенд
│   ├── src/
│   │   ├── components/    # Переиспользуемые компоненты
│   │   ├── pages/        # Страницы
│   │   ├── layouts/      # Шаблоны страниц
│   │   ├── hooks/        # Кастомные хуки
│   │   ├── services/     # API сервисы
│   │   └── utils/        # Утилиты
│   └── package.json
├── server/                # Бэкенд
│   ├── src/
│   │   ├── controllers/  # Контроллеры
│   │   ├── models/       # Модели данных
│   │   ├── routes/       # Маршруты
│   │   ├── middleware/   # Промежуточные обработчики
│   │   └── utils/        # Утилиты
│   └── package.json
└── README.md
```

## Технологии

- **Фронтенд**: Vanilla JavaScript без фреймворков, Vite для сборки
- **Бэкенд**: Node.js с собственной реализацией сервера

## Запуск проекта

### Клиентская часть

```bash
cd client
npm install
npm run dev
```

### Серверная часть

```bash
cd server
npm install
npm run dev
```

## Особенности

- Собственная реализация компонентной архитектуры
- Роутинг на чистом JavaScript
- Динамическое обновление информации о сервере
- Адаптивный дизайн
