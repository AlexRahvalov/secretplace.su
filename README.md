# SecretPlace.su - Сайт для Minecraft сервера

Проект сайта для Minecraft сервера с динамически обновляющейся информацией, поддержкой множества серверов и оптимизированной производительностью.

## Основные возможности

- Мониторинг статуса нескольких Minecraft серверов
- Система кэширования для оптимизации производительности
- Адаптивный пользовательский интерфейс
- Интеграция с различными системами авторизации
- История статистики серверов
- Интеграция с Telegram для авторизации и уведомлений

## Технологии

- **Фронтенд**: Vanilla JavaScript, Vite для сборки
- **Бэкенд**: Node.js, Express.js
- **База данных**: MariaDB (основная), SQLite (для локальной разработки)
- **Кэширование**: Redis
- **Аутентификация**: Собственные адаптеры для EasyAuth, AuthMe, Telegram

## Оптимизация производительности

- Универсальный менеджер кэширования с TTL
- Кэширование часто запрашиваемых данных
- Оптимизация запросов к базе данных
- Агрегация статистики по временным интервалам
- Параллельное обновление статусов серверов

## Запуск проекта

### Установка всех зависимостей

```bash
npm run install:all
```

### Запуск в режиме разработки

```bash
npm run dev
```

### Запуск только клиентской части

```bash
npm run client
```

### Запуск только серверной части

```bash
npm run server
```

## Архитектура проекта

### Клиентская часть

- **Компонентная архитектура**: Компоненты, реализованные на чистом JavaScript
- **Роутинг**: Собственная реализация роутинга на клиенте
- **Сервисы**: Отдельные сервисы для работы с API

### Серверная часть

- **MVC**: Структура Model-View-Controller
- **API**: RESTful API для взаимодействия с клиентом
- **Адаптеры**: Система адаптеров для различных способов аутентификации
- **База данных**: Поддержка различных СУБД через абстракцию
- **Кэширование**: Двухуровневая система кэширования (память + Redis)
