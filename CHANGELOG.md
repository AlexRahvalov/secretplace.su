# Журнал изменений (Changelog)

## [0.0.0.0] - 2023-03-20

### Добавлено
- Разработана клиентская часть на чистом JavaScript с использованием Vite
- Реализована собственная компонентная система
- Создан роутер на основе History API
- Добавлены динамические компоненты Header и Footer
- Разработана главная страница с информацией о сервере
- Добавлена страница ошибки 404 (Not Found)
- Создан сервер на Node.js с использованием встроенного http-модуля
- Реализованы API эндпоинты для получения информации о сервере и новостей
- Настроены прокси для взаимодействия клиента и сервера
- Добавлены скрипты для удобного запуска приложения

### Технические детали
- **Версия проекта**: 0.0.0.0
- **Версия клиента**: 0.0.0.0
- **Версия сервера**: 0.0.0.0

### Структура проекта
- Создана базовая структура проекта с разделением на клиентскую и серверную части
- Добавлены все необходимые директории согласно требованиям
- Настроены базовые стили и адаптивный дизайн
- Настроена работа с динамически обновляющимися данными
